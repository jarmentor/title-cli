# APA Title-CLI

Convert a string or file of strings to APA style title case.

## Installation

```bash
  npm install @jarmentor/title-cli
```

## Usage

### Via NPX

```bash
  npx @jarmentor/title-cli <string OR file> [options]
```

### After installation

```bash
  title-cli <string> [options]
```
